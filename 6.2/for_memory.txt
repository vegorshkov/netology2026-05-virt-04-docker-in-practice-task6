Итог метода

Используем только docker build и Dockerfile

Копируем файл /bin/terraform внутрь нового слоя /output

Затем извлекаем его через docker cp

Бинарник работает корректно

Примечание: альтернативный метод — использовать FROM hashicorp/terraform и COPY --from=0 /bin/terraform ./terraform в многоступенчатом Dockerfile (multi-stage build) для прямого извлечения в локальную директорию при сборке, но вышеописанный способ проще для учебного задания.
